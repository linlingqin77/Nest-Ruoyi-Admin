## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Unit testing coverage

- 所有业务模块（system、monitor、common、upload、backup、main）现已提供 Jest 单元测试，测试文件位于相应模块目录下的 `*.service.spec.ts`。
- 如使用 npm，可运行 `npm run test -- --runInBand` 以串行执行全部单测，确保与数据库/Redis 交互 mock 保持稳定。
- 若希望观察特定模块，可使用 `npx jest src/module/system/system.services.spec.ts` 这类命令按需运行。
